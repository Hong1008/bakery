# 시장조사 및 구매관리
## 1. 시장조사
- 목적: 예산 결정, 합리적 구매 계획, 신제품 설계, 제품 계량
- 시장조사 원칙: 비용 경제성, 조사(적시,탄력, 계획, 정확)
- 폐기율 <> 가식부
- 폐기순서: 어패 > 육류 > 채소/과일 > 서류 > 달걀 > 유지 > 해조 > 두류 > 곡류
## 2. 식품 구매
- 대량 또는 공동 구매로 저렴하게 구입
- 가격과 출회표에 유의
- 육류는 냉장 시 일주일분
- 과일 채소 어패류는 선도에 따라 수시로 구입
- 곡류, 건어물, 조미료는 1개월분
- 비가식부와 폐기율을 고려
- 절차: 발주 -> 납품 -> 검수
- 구매명세서 포함 내용: ~, 숙성 정도, 전처리 및 가공 정도, 보관 온도, 폐기율
- 발주량 계산식
	- 총발주량: 정미중량 * 100 / (100-폐기율) * 인원수
	- 필요비용: 필요량 * 100 / 가식부율 * 1kg 단가
	- 대체 식품량: 원래 식품의 양 * 원래식품의 성분 수치 / 대체 식품 성분 수치
## 3. 재고관리
- 재고회전율: 높을수록 고갈
- 재고 평가 방법
	- 선입선출
	- 후입선출: 재고자산 평가 최소화
	- 개별법: 단가별 가격
	- 단순평균법: 일정기간 동안 구입 횟수로 나눈 구입단가의 평균을 소비단가로 함
	- 이동평균법
- 당기소비량 = (이월 + 당기) - 기말 재고량
- 재료 소비량 = 표준소비 * 생산
# 검수관리
- 검수 유의 사항
	- 검수기기는 당도계, 염도계등을 준비, 도착하자마자 검사
	- 배송 차량의 청결, 내부온도 점검
	- 검수대 바닥에서 60cm이상에서 검수
	- 맨손 관능 검수 x
	- 540lux
	- 규격에 안맞으면 반품
	- 김치류는 관능검사
	- 얼음이나 물은 제거 후 수량/중량 검수
	- 소비기한 확인
- 전수 검사법: 하나하나 전부 검사
- 발췌 검사법(샘플링): 일부 뽑아 검사
- 검수 순서: 냉장 -> 냉동 -> 신선 -> 공산
- 검수 절차: 서류 대조 -> 품질 검사 -> 인수/반품 -> 입고 -> 검수 기록 문서 정리

## 1. 식품감별
1. 쌀: 낟알 상태, 건조 상태, 이물질 여부, 산지, 수확 시기
	1. 타원형
	2. 싸라기 적고 이물질 없음
	3. 광택 있고 입자가 고름
	4. 건조 잘 되어있는 것
2. 밀가루: 색과 광택
	1. 희고 밀기울 없음
	2. 뭉쳐있지 않고 미세함
	3. 잘 건조 된 것
3. 수산식품과 그 가공품: 신선도, 외관
	1. 신선한 어류는 물에 가라 앉음
	2. 윤이나고 광택이 있고 비늘이 고르게 밀착
	3. 살이 탄탄하고 탄력성 존재
	4. 눈이 투명하며 돌출됨, 아가미 선홍색
4. 육류
	1. 소고기는 선홍색과 윤기, 돼지는 분홍색을 띠는 붉은 색
	2. 윤기가 나고 탄력
	3. 병육은 피가 많아 냄새남
5. 채소류
	1. 양배추: 단단하고 무거움
	2. 배추: 굵은 섬유질이 없고 속에 심이 없음
	3. 오이: 굵기가 고름, 가시가 있고 무거움
	4. 대파: 흰대가 굵고 긺
	5. 깻잎: 잎이 마르지 않음
6. 달갈: 껍질이 까칠하고 광택 없음, 흔들었을때 소리 안남
7. 우유
	1. 점성이 없고 100도 까지 응고 안함
	2. 물 컵에 우유 떨어뜨리면 구름처럼 퍼짐
# 원가
- 목적: 가격 결정, 원가 관리, 예산 편성, 재무제표 작성
- 매월 계산이 원칙이지만 3개월 또는 1년에 한번 하기도 함
- 원가 구성도

| 직접원가 | 제조간접비 | 제조원가  | 총원가                 | 판매가격 |
| ---- | ----- | ----- | ------------------- | ---- |
| 재료   | 재료    | 직접원가  | 판매관리비(판매경비 + 일반관리비) | 이익   |
| 노무   | 노무    | 제조간접비 | 제조원가                | 총원가  |
| 경비   | 경비    |       |                     |      |
- 원가 계산 원칙: 부정적인 것이 없음, 검수가 없음
- 단계: 요소 -> 부문 -> 제품
- 손익분기점 = 고정비 + 변동비
